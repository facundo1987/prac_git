# Trabajo Colaborativo con Git y GitHub

## Descripción

Este repositorio fue utilizado para desarrollar la actividad práctica de Trabajo Colaborativo con Git y GitHub de la UTU.

Durante la práctica se realizaron las siguientes actividades:

- Creación y gestión de ramas.
- Trabajo colaborativo entre los integrantes.
- Registro de cambios mediante commits.
- Integración de ramas con merge.
- Sincronización con GitHub mediante push y pull.
- Resolución de conflictos de fusión.

## Herramientas utilizadas

- Git
- GitHub
- Git Bash
- Visual Studio Code

## Objetivo

Aplicar los conceptos básicos de control de versiones utilizando Git y GitHub en un entorno de trabajo colaborativo.
